# CyroTime Pins

This is the list of used pins .



## Physical Pins Used by devices

### Dot-Matrix display pins

```
* DATA_PIN ---> 23
* CLK_PIN ---> 18
* CS_PIN ---> 5
```	
VCC Pin to 5 Volt. And GND pin to Ground .

### LDR, IR & Light Pin..

```
* LDR_PIN ---> 34
* IR_OUT ---> 25
* LIGHT_PIN ---> 27
```
LDR & IR Input has been grounded via 10k Ohm Resistor.
